# AI Decision Systems

This repository outlines how I design and build AI-powered decision systems that translate real-world signals into operational actions.

## What I Build
- Decision intelligence systems (not just AI copilots)
- Signal → reasoning → action pipelines
- Hybrid systems combining LLMs + deterministic business logic
- Production-ready AI workflows with reliability and guardrails

## Architecture Principles

### 1. Signal Ingestion
Structured + unstructured inputs:
- Sales, inventory, behavioral data
- Environmental and contextual signals
- External APIs (weather, market signals, etc.)

### 2. Reasoning Layer
- LLMs (Gemini, Claude, OpenAI)
- Prompt orchestration
- Controlled reasoning with constraints

### 3. Decision Layer
- Deterministic logic + scoring
- Risk evaluation
- Action prioritization

### 4. Execution Layer
- APIs
- UI / product integration
- Automated workflows

## Example Use Cases
- Retail inventory allocation (Hebra AI)
- Contextual real-world intelligence (Wagamour)
- Signal-driven product recommendations

## Stack
- GCP (Cloud Run, BigQuery, Vertex AI)
- Gemini, Claude, OpenAI APIs
- Cursor for AI-assisted development
